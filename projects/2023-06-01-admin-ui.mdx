This application is a Content Management System (CMS) designed to manage corporate portal pages,
 numbers, functionality, and data. It provides a comprehensive Admin UI with various features 
 to streamline the management of the corporate portal.

## Features
- Add, edit, and remove departments, teams, and positions.
- Create and manage users and roles.
- Customize the content of each page.
- Set up key performance indicators (KPIs) to display on the portal.
- Create rich text articles using CKEditor 5.
- Manage image galleries.
- Moderate and manage comments to maintain a healthy discussion environment.
